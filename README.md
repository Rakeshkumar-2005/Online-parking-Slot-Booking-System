# 🚗 Online Parking Slot Booking System

An **Online Parking Slot Booking System** is a web-based application that allows users to search, book, and manage parking slots online. The project aims to reduce the time spent searching for parking spaces and provide a convenient parking experience.

---

## 📌 Features

- 👤 User Registration & Login
- 🔍 Search Nearby Parking Locations
- 🅿️ View Available Parking Slots
- ✅ Book Parking Slot
- 💳 Payment Page (Demo)
- 🎉 Booking Confirmation
- 📊 User Dashboard
- 🛠️ Admin Dashboard
- 📞 Contact Page
- ℹ️ About Page
- ⭐ Reviews Page
